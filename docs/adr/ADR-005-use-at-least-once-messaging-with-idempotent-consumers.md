# ADR-005: Use at-least-once messaging with idempotent consumers

Status: Accepted — Retrospective record
Original decision date: 13 July 2026
Authority: LedgerOps Technical Specification v1.0

## Provenance

This file is a retrospective record of a decision originally approved in LedgerOps Technical Specification v1.0 on 13 July 2026. It is not a recovered original and introduces no new decision.

## Decision

Use at-least-once message delivery with idempotent consumers and do not claim end-to-end exactly-once delivery.
